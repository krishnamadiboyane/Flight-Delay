# Flight Delay Prediction

This project predicts flight delays using Machine Learning techniques based on historical flight data.

## 📌 Objective
To analyze flight delay patterns and build a predictive model that estimates delays using past data.

## 📂 Dataset
- atc_pre_departure_delays_2024.csv

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas, NumPy
- Machine Learning algorithms

## 📊 Files Included
- Flight_Delay_Prediction.ipynb → Model building and analysis  
- atc_pre_departure_delays_2024.csv → Dataset  
- flight delay prediction Output screenshots.rar → Output results
